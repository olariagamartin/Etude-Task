# Etude-Task
Simple To Do task with some features 

* Create projects
* Add tasks
* Add sub-tasks
* Notes
* Due date
* Notifications
* Dark Theme

---

Try the app: https://play.google.com/store/apps/details?id=com.themarto.etudetask
